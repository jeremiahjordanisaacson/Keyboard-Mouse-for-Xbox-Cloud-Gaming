# Privacy Policy for Keyboard & Mouse for Xbox Cloud Gaming

**Last updated: January 2025**

## Overview

"Keyboard & Mouse for Xbox Cloud Gaming" is a browser extension that enables keyboard and mouse controls for Xbox Cloud Gaming (xCloud). This privacy policy explains how we handle your data.

## Data Collection

**We do not collect any personal data.**

This extension:
- Does NOT collect any personal information
- Does NOT track your browsing activity
- Does NOT send any data to external servers
- Does NOT use analytics or tracking
- Does NOT access any data outside of xbox.com/play

## Data Storage

The extension stores the following data **locally on your device only**:
- Your control settings (mouse sensitivity, key bindings, etc.)
- Whether the extension is enabled or disabled

This data is stored using your browser's built-in storage API and **never leaves your device**.

## Permissions Explained

The extension requests these permissions:

1. **storage** - To save your settings locally on your device
2. **activeTab** - To check if you're on the Xbox Cloud Gaming page
3. **Host permissions for xbox.com** - To inject the virtual gamepad code that enables keyboard/mouse controls

## Third-Party Services

This extension does not use any third-party services, analytics, or external APIs.

## Data Sharing

We do not share any data with anyone because we do not collect any data.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top.
