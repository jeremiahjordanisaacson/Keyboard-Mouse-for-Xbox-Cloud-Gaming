Features

WASD

Full Keyboard Support

Use any key on your keyboard to control movement, actions, and menus. WASD moves, Space jumps, and everything is customizable.

+

Multi-Key Bindings

Bind multiple keys to the same action. Use both Space and Enter for the A button, or create comfortable alternatives.

Aim

Mouse Aiming

Precise mouse control for camera and aiming. Click in the game to lock your cursor and start aiming.

Profiles

Game Profiles

Create different profiles for different games. Auto-switch profiles when you launch specific games.

Curves

Sensitivity Curves

Choose linear, exponential, or S-curve sensitivity. Adjust deadzone for perfect control feel.

Macro

Macro Recording

Record button sequences with timing and replay them with a single key press. Perfect for combos.

Cloud

Cloud Sync

Your settings sync across devices via Chrome. Set up once, play anywhere with the same configuration.

Overlay

On-Screen Overlay

See your current profile and status with a minimal overlay. Drag to reposition or hide completely.

How It Works

1

Install Extension

Add the extension to Chrome or Edge from the browser store. It's free and open-source.

2

Open Xbox Cloud Gaming

Go to xbox.com/play and start any game. The extension activates automatically.

3

Click to Lock Mouse

Click on the game stream to lock your mouse for camera control. Press Escape to unlock.
